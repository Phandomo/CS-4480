############################
#                          #
#    Setup Instructions    #
#                          #
############################

1. Extract the "pa2_b.tar" file and verify that it produced eight .java files:

    Event.java
    EventList.java
    EventListImpl.java
    Message.java
    NetworkSimulator.java
    Packet.java
    Project.java
    StudentNetworkSimulator.java
	
2. Execute the following command to compile the eight classes with version 7 of the JDK:

    /usr/local/apps/jdk/jdk1.7.0_45/bin/javac *.java


############################

    /usr/local/apps/jdk/jdk1.7.0_45/bin/java Project
